25.31 java.lang.InstantiationException (JDK 1.0)

Signals an attempt to instantiate an interface or an abstract class. 

public class InstantiationException extends Exception {
    // Public Constructors
            public InstantiationException();
            public InstantiationException(String s);
}

Hierarchy:

Object->Throwable(Serializable)->Exception->InstantiationException

Thrown By:

Class.newInstance(), Constructor.newInstance()
