18.11 java.awt.CheckboxGroup (JDK 1.0)

A CheckboxGroup object enforces mutual exclusion (also known as "radio button" behavior) among any number of Checkbox buttons. A Checkbox component can specify a CheckboxGroup object when created or with Checkbox.setCheckboxGroup(). If a Checkbox with a given CheckboxGroup object is selected, then the CheckboxGroup ensures that the previously selected Checkbox becomes unselected.
